# Sagheer's Portfolio Website

Welcome to my personal designer & developer portfolio. This project serves as a showcase of my design work (built in Figma) and my development experiments (built with code).

## Features
- **Fully Static & Fast**: Built purely with semantic HTML5, CSS3 variables, and vanilla JavaScript.
- **Premium Design**: Modern, highly polished dark mode interface with glassmorphism effects, neon cyan/pink accents, and beautifully paired typography (Inter & Outfit).
- **Subtle Animations**: Uses Intersection Observer for buttery-smooth scroll reveals, custom CSS-driven lift & glow hover states, and dynamic pure-CSS art in the Lab section.
- **Zero Dependencies**: No bloated node modules, frameworks, or complex build steps.
- **Cloudflare Ready**: Optimized perfectly for one-click, secure static deployment to Cloudflare Pages.

## Security Overview
This codebase is completely public-ready. There are no backend requirements, `.env` files, API keys, or embedded secrets.

## Usage
Simply clone or download this repository and link it to Cloudflare Pages (or drag the directory into the dashboard). 

## Local Development
To run this project locally, you can use any static server, or the VSCode **Live Server** extension.
